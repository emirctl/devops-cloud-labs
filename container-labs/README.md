# Docker Lab

## Overview
This repository demonstrates running Nginx inside a Docker container to serve
static content.

The project focuses on:
  -  Dockerfile basics
  -  Nginx configuration
  -  Port mapping with Docker
  -  Clean and minimal structure

# Requirements
  -  Git
  -  Docker
  -  Docker Compose
